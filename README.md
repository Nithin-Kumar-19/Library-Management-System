# Library Management System
A simple command-line based Library Management System in Python, demonstrating core Object-Oriented Programming (OOP) principles. Features include book and member management, borrowing and returning books, searching by title and author, and displaying library data interactively.

# Features
Add, remove, search, and display books

Register and remove members

Borrow and return books

Search books by title or author (case-insensitive, partial match)

View all borrowed books and members’ borrowed lists

Interactive menu-driven terminal interface

# Usage
You’ll see a menu with numbered options:

Add Book: Add new books to the library

Remove Book: Remove books (if not borrowed)

Add Member: Register new members

Remove Member: Remove members (if no books are borrowed)

Search Books: Find books by title or author

Borrow/Return Book: Manage borrowing status for members

Display Books/Members/Borrowed Books: View current state of the library

Exit: Quit the application
